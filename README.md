Automation Car Service Solutions:
An end-to-end web-based automation system designed to streamline car service booking,
management, and administration.

Project Overview:
Automation Car Service Solutions is a full-stack web application that enables customers to book car
services online while allowing administrators to manage users and service requests efficiently.

Key Features:
• User registration and secure login
• Online car service booking
• Admin dashboard for managing services
• Database-driven service management

Tech Stack:
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL (PDO)
Tools: XAMPP, VS Code, GitHub

System Architecture:
User Interface → PHP Backend → MySQL Database → Admin Dashboard

How to Run
1. Install XAMPP
2. Create database 'automatedcarservices'
3. Configure db_connect.php
4. Run on localhost via browser

• Online payments
